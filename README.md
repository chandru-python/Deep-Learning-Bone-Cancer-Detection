# 🦴 Bone Cancer Detection using CNN and Flask

## 📌 Description
This project implements a **Bone Cancer Detection System using Convolutional Neural Networks (CNN)**. The model classifies medical images into **malignant or normal categories**. A **Flask-based web application** allows users to upload bone images and get instant predictions through a simple interface.

## ✨ Features
- 🤖 Deep Learning based image classification  
- 🧠 CNN model for bone cancer detection  
- 🌐 Web interface using Flask  
- 📤 Image upload and prediction system  
- 🔐 User login and registration  
- 🗄️ MySQL database integration  

## 🛠️ Technologies Used
- 🐍 Python  
- 🔬 TensorFlow / Keras  
- 🌐 Flask  
- 🗄️ MySQL  
- 🎨 HTML / CSS  
- 📊 NumPy  
- 📷 OpenCV  

## 📂 Dataset
The dataset contains bone X-ray images categorized into:
- ⚠️ Malignant  
- ✅ Normal  

Images are stored in separate folders for training the CNN model.

## 🧠 Model Architecture
The CNN model consists of:
- Convolutional Layers  
- ReLU Activation  
- Max Pooling Layers  
- Fully Connected Dense Layers  
- Softmax Output Layer  

## ⚙️ Installation
1. Clone the repository  
git clone https://github.com/yourusername/bone-cancer-detection.git  

2. Navigate to the project folder  
cd bone-cancer-detection  

3. Install required libraries  
pip install -r requirements.txt  

## ▶️ Run the Application
Start the Flask server  
python app.py  

Open your browser and go to  
http://127.0.0.1:5000  

## 🚀 Usage
1. Register or login to the system  
2. Upload a bone X-ray image  
3. The CNN model analyzes the image  
4. The system displays whether the image is **Malignant or Normal**
